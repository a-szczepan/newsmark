#### Newsmark
##### Description
Newsmark is a demo project whose main functionality is to allow users to browse, search and read articles published on The New York Times website. The user has the option to set a *bookmark* to save the article to the user's list. In addition, by selecting a section of text from an article, the user can *annotate* it with a personal note. The selected text snippet is highlighted and, by clicking on it, the user's note is opened. Notes can be edited and deleted. From the user page, it is possible to view and search all bookmarked articles and their notes.
###### Technologies, libraries, frameworks
**Frontend**: JavaScript / TypeScript / Sass (SCSS) + CSS Modules / React.js / Redux Toolkit / React Router / Storybook / Webpack\
**Backend**: JavaScript / Node.js / Express.js / MySQL / Sequelize / OAuth / JWT
